# 院辦Email催繳
無聊寫的😜

# 特色
* 根據模板的文字格式產生內容
* 可根據表單內容替換文字
* 連動Outlook一鍵發送郵件，不用擔心格式跑掉
* 自動附加文件，不怕傳錯檔案

# 使用方法
### 基本操作
1. 填入資料
2. 按下"生成文字"

3. 按下"傳送郵件"
4. 發送完成！！
